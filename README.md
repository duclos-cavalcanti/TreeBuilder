## 1. Setup

1. Clone the repo
   ```bash
   git clone --recursive https://github.com/duclos-cavalcanti/msc-thesis.git
    # recursive flag needed to include submodule
   ```

2. Install submodules.
   ```bash
   printf 'Hello World\n'
    ```

1. Build
    ```bash 
    make build
    ```

